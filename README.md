# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Running the app

```bash
# Development
npm run dev

# Preview (serve the /dist directory)
npm run preview
```

## Building the app

```bash
npm run build
```

## Tooling

```bash
# Lint
npm run lint
```
